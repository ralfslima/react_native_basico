Microsoft Excel

O Microsoft Excel ou simplesmente Excel, é um dos softwares mais requisitados no mercado de trabalho, com ele é possível criar planilhas detalhadas para auxiliar no seu trabalho, além de fórmulas que facilitam muito no seu trabalho, contamos com diversas funcionalidades como: gráficos, tabelas dinâmicas, dashboards, importação de dados e muito mais.

Na Campos Treinamentos você encontra os três níveis de Excel ou pode optar pela formação completa:

  • Excel Básico
  • Excel Intermediário
  • Excel Avançado
  • Formação Excel

Microsoft Power Point

O Microsoft Power Point ou simplesmente Power Point, é um software desenvolvido para criar apresentações. Desde como planejar uma apresentação até trabalhar com efeitos de transições, gravações e modo de apresentações, você será capaz de trazer aquele efeito UAU em suas apresentações.

Crie apresentações profissionais, a Campos Treinamentos possui um treinamento completo de Power Point, entre em contato conosco para mais informações.

Aprenda a utilizar: Sumários, rodapé, ilustrações, etiquetas, e até ditado.
